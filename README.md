# Part 1: Data Audit, EDA & Business Understanding

## What this repo contains
- `eda_audit.ipynb` — Full EDA notebook with all analysis, charts, and hypotheses
- `data_quality_report.md` — Summary of quality issues found across all 7 datasets
- `business_memo.md` — Business-facing memo with actionable findings
- `charts/` — Saved chart images from the notebook

## How to run

1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Place raw CSV files in `../data/raw/` or update paths in the notebook
4. Run all cells in `eda_audit.ipynb`
